This version is for MacOS Intel 10.5

To use the Oracle Instant Client it is necessary to set the ORACLE_HOME and DYLD_LIBRARY_PATH to the right directory, 

e.g.
DYLD_LIBRARY_PATH=/sw/oracle
ORACLE_HOME=/sw/oracle
PATH=/usr/local/bin:/bin:/sbin:/usr/bin:/usr/sbin:/sw/bin:/sw/oracle
TNS_ADMIN=/sw/oracle


Then run checkpwd.
